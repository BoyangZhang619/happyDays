- 内存优化，自动清理屏幕外雪花

### 🎁 智能装饰品选择系统
- **网格布局**：16种圣诞装饰品整齐排列

- 支持四种语言的专属祝福风格
- 100-200字精美祝福语

## 🎯 使用方法

### 1. 在线访问

直接访问部署在 Vercel 的在线版本：[https://happy-days-rho.vercel.app](https://happy-days-rho.vercel.app)

### 2. 本地运行

没法本地运行


### 3. 操作指南

1. **选择语言**：点击页面顶部的语言选项（中/英/法/日）
2. **选择装饰品**：在网格中点击圣诞装饰品进行选择
   - 点击次数遵循：1个→0个→2个→0个→3个→0个... 的循环
   - 装饰品会显示当前选择数量
3. **生成祝福语**：选择完装饰品后，点击"生成圣诞祝福语"按钮
4. **查看祝福**：AI会根据您选择的装饰品和语言生成专属祝福语

### 4. 快捷键

- `Ctrl + Enter`：快速生成祝福语

### 5. 移动端使用

- **触摸操作**：轻触装饰品进行选择，避免长按或拖拽
- **页面滚动**：在装饰品之间的空白区域滑动进行滚动
- **语言切换**：轻触语言选项即可切换
