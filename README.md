# 🎄 圣诞AI祝福生成器 🎅

一个充满圣诞节魔力的AI祝福语生成器，通过收集散落在页面上的圣诞装饰品来获得专属的AI生成祝福语！

## ✨ 主要功能

### 🌨️ 动态雪花效果
- 随机生成各种样式的雪花（❄️ ❅ ❆ ✻ ✼ ❋ ❊ ⛄ ❉ ❈）
- 雪花以不同速度和方向飘落
- 具有随机的左右摇摆和旋转效果
- 自动内存管理，超出页面的雪花会被清理

### 🎁 装饰品收集系统
- **主容器装饰品**：在主要区域展示的装饰品，点击选择
- **散落装饰品**：随机散布在整个页面的小装饰品，点击收集
- **16种圣诞装饰品**：
  - 🎄 圣诞树、🎁 礼品盒、🍭 拐杖糖、🧦 圣诞袜
  - 🎅 圣诞帽、🔔 铃铛、⭐ 星星、⛄ 雪人
  - 👼 天使、🕯️ 蜡烛、🎀 花环、🍪 姜饼
  - 🦌 驯鹿、🛷 雪橇、🧤 手套、🧣 围巾

### 🤖 AI祝福语生成
- 基于收集的装饰品类型和数量生成个性化祝福语
- 调用阿里云通义千问API
- 融合每个装饰品的美好寓意
- 生成温馨、富有诗意的圣诞祝福

### 🎨 精美动画效果
- **装饰品动画**：悬浮、点击、选中效果
- **按钮动画**：渐变光效、悬停缩放
- **文字效果**：彩虹渐变标题、闪烁边框
- **收集特效**：点击时的星光特效、+1浮现动画
- **页面元素**：入场动画、呼吸效果

## 🎯 使用方法

### 1. 本地运行
```bash
# 方法1：使用Python
cd d:\gitLocal\happyDays
python -m http.server 8000

# 方法2：使用批处理文件（Windows）
双击运行 start_server.bat

# 方法3：使用Node.js
npx http-server -p 8000
```

然后访问：`http://localhost:8000`

### 2. 操作指南
1. **收集装饰品**：点击页面上散落的圣诞装饰品进行收集
2. **选择装饰品**：在主容器中点击装饰品来选择/取消选择
3. **生成祝福语**：收集足够装饰品后，点击"生成圣诞祝福语"按钮
4. **查看祝福**：AI将根据您的装饰品组合生成专属祝福语

### 3. 快捷键
- `Ctrl + Enter`：快速生成祝福语
- `Ctrl + R`：重新排列装饰品位置并清空选择
